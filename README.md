# todoList

# About the Project

A TodoList Java app, which is deployed over Heroku. 
https://tranquil-ocean-66378.herokuapp.com/



## Tools and Technology Used
   - Java 8
   - Spring Boot
   - Maven
   - Thymeleaf (for ui)
   - BootStrap
   - Git CLI
   - Mysql Database
   - H2 Database
